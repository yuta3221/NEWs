# simple-chat

## タイトル
```
シンプルチャットアプリ

サイトリンク　　https://chat-vue-app-fce21.web.app/
```

### 概要
```
リアルタイムでコメントリアルタイムでコメントが同期されるアプリです。
```

### 機能
```
・コメントの追加
・コメントの削除
が可能です。
その他の機能を随時追加していきます。
```

### アピールポイント
```
別プラウザでwebアプリを開いて、どちらかでおこなった操作が別の方にもリロードなしで、
同期されます。
```

### 使用技術
・Vue.js
・Firebase
・vuetifyjs
・Firestore
